# Djangoで作ったTODOアプリ（練習用）

Djangoの練習で作ったスクール課題の初Webアプリです。 <br>
無事にCRUD（作成・表示・編集・削除）全てが実装できました。

## できたこと
* **タスクの登録**： 「重要」の文字を入れると文字数(5文字以上)の指定が表示される
* **ステータスの切り替え**： 完了・未完了をボタンで変えられる
* **タスクの編集と削除**： 【DeleteView】や【UpdateView】使用

## 苦労したところ（備忘録）
* **NoReverseMatch**： リンクを作るとき【task.id】を渡し忘れて画面が黄色になったり、エラー多発
* **URLのつなぎこみ**： 【urls.py】と【views.py】の連携が最初全然わかず、何回もやり直し
* **URLの変数**： どのタスクを編集するかURLに教えるため update/<int:pk>/ と書く部分が難しかった

## 使い方
1. 【python manage.py runserver】起動
2. 【http://127.0.0.1:8000/todo/list/】アクセス
